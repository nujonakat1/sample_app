Twitterを模した投稿アプリです。
（Ruby on Rails チュートリアルのサンプルアプリケーションです）

レスポンシブ対応しているのでスマホからもご確認いただけます。

URL
https://sample-app-1zmk.onrender.com

機能の一覧
・ユーザー登録、ログイン機能
・140文字の投稿機能
・画像投稿
・フォロー機能

使用技術
・Ruby on Rails 7.0.4
